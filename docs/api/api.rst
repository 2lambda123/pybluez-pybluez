
*****************
API Documentation
*****************

This document contains the API (Application Programming Interface) documentation for PyBluez. 
Documentation for the Python objects defined by PyBluez is divided into separate pages for each 
package, module, and class.

Object Documentation
""""""""""""""""""""

Each **Package Documentation** page contains:

-  A description of the package.
-  A list of the modules contained by the package.


Each **Module Documentation** page contains:

-  A description of the module.
-  A summary of the functions defined by the module.
-  A summary of the classes defined by the module.

Each **Function Documentation** page contains:

- A detailed description of the function defined by the module.


Each **Class Documentation** page contains:

-  A description of the class.
-  A summary of the methods defined by the class.
-  A summary of the class (static) attributes defined by the class.

Each **Method Documentation** page contains:

-  A detailed description of the method defined by the class.
